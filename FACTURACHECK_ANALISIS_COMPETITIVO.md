## 🎯 DECISIONES (Confirmadas por investigación)

### DECISIÓN 1: PAC = Facturama ✅
**Razón**: 
- Costo bajo ($55-1,650/año vs no-público CONTPAQi)
- API simple + sin paquetes previos (modelo pago por uso perfecto)
- Integración webhook para saldo + consumo
- Best-in-class para PyMEs/medianas (nuestro target)
- No es "único" PAC, pero mejor para FacturaCheck propósito

**Alternativa rechazada**: CONTPAQi es enterprise (caro + complejo), Aspel es instalable (overhead)

---

### DECISIÓN 2: Compra de Timbres = EN nuestro dashboard ✅
**Razón**: Tu requisito ("muy sencilla")

**Opción A rechazada**: Usuario compra en Facturama (fragmented UX)  
**Opción B confirmada**: Compra EN FacturaCheck
- Saldo prepagado + destajo
- Nosotros cobramos, reconciliamos con Facturama via webhook
- Línea sobregiro para urgencias
- UX centralizada + oportunidad monetización

---

### DECISIÓN 3: Distribución = Email + WhatsApp + Descarga ✅
**Hallazgo**: **Ningún competidor envía por WhatsApp automáticamente** → BRECHA

**Implementar**:
- Email automático (emisor + receptor) ✅
- WhatsApp automático (emisor + receptor) ✅ **DIFERENCIAL**
- Descarga archivo ZIP (XML + PDF) ✅
- Historial de envíos

**Validación**: Contador + usuario aprueban, después auto-envío

---

### DECISIÓN 4: Integración CobraCheck = NATIVA ✅
**Hallazgo**: CONTPAQi tiene integración Cobra pero costo alto, NO nativa  
**Nuestro**: Trigger cobro pagado → auto-genera CFDI

**Campos CobraCheck requeridos**:
- `client_rfc` (obligatorio)
- `payment_amount` (obligatorio)
- `payment_method` (obligatorio)

**Validación**: Sincronización 1:1, status sync (si se cancela cobro → cancela CFDI)

---

### DECISIÓN 5: Otros Módulos = GastoCheck + BancoCheck ✅
**GastoCheck**: Gasto aprobado → CFDI egreso automática  
**BancoCheck**: CFDI vinculada a transacción bancaria, reconciliación automática

**Validación**: Todas sincronizadas, no fragmentadas

## 📋 RESULTADOS DETALLADOS

### ✅ Facturama
- **Modelo**: SaaS 100% cloud. Sin paquetes previos
- **Pricing**: $55/año (10 folios), $165/año (40 folios), $1,650/año (ilimitado). Extras $0.40-0.50/folio
- **Características**: Timbrado ilimitado (modelo), masivo por Excel, Shopify $13 USD/mes, Nómina integrada
- **Distribución**: Email ✅, almacenamiento cloud ✅, WhatsApp ❌ (solo soporte)
- **Integración Banca**: ❌ No
- **Auditoría 5Y**: ⚠️ Cloud storage (vago)
- **Diferencial**: Entrada muy baja ($55), masivo, muchas integraciones B2B
- **Debilidad**: Sin integración bancaria, no apto para medianas/grandes

### ✅ CONTPAQi CFDI
- **Modelo**: SaaS 100% cloud. PAC histórico (1er autorizado por SAT)
- **Pricing**: Precios no públicos, suscripción anual con timbrado incluido
- **Características**: CFDI 4.0, Complementos (Retenciones 2.0, Pagos, Nómina), Digital Documents Manager
- **Distribución**: Email ✅, portal web ✅, WhatsApp ❌
- **Integración Banca**: ✅ **LÍDER.** CONTPAQi Bancos + Banorte BEM. Conciliación automática. CFDIs pago/retención
- **Auditoría 5Y**: ✅ Digital Documents Manager. Validación + almacenamiento + consulta. Immutable audit trail
- **Integración Cobra**: ✅ CONTPAQi Cobra (CxC), sync con CFDI, DIOT automática
- **Diferencial**: Ecosistema integral (Contabilidad+Cobra+Nómina+Bancos). B2B robusto. Auditoría enterprise
- **Debilidad**: Pricing no público, caro, complejo para PyMEs

### ✅ Aspel-COI v11
- **Modelo**: Hybrid. Software instalable + PAC para timbrado
- **Pricing**: Costo inicial alto (versión/módulo dependiente)
- **Características**: CFDI 4.0, Complementos (Retenciones, Pagos, Nómina), Auditoría de pólizas immutable
- **Distribución**: Email ✅, XML ✅, WhatsApp ❌
- **Integración Banca**: ✅ Aspel-Banco + COI. Movimientos→pólizas automático
- **Auditoría 5Y**: ✅ Depósito de Documentos automático. Auditoría de pólizas impide modificaciones
- **DIOT**: ✅ Automática desde pólizas dinámicas
- **Diferencial**: Formal, histórico, mejor para contadores/despachos. Auditoría enterprise-grade
- **Debilidad**: Instalable (overhead), no SaaS, caro

### ✅ Siigo
- **Modelo**: SaaS 100% cloud. Posicionamiento DIAN (Colombia), penetración débil MX
- **Pricing**: $0 (5 CFDIs free), $1,236 MXN/usuario/año (ilimitado)
- **Características**: Facturación ilimitada. POS integrado. Análisis CFDIs
- **Distribución**: Email ✅, portal web ✅, WhatsApp ❌
- **Integración Banca**: ⚠️ Limitada. Mencionada pero no detalles
- **Auditoría 5Y**: ⚠️ Sincronización SAT mencionada, pero específico no claro
- **Diferencial**: Modelo freemium generoso. POS included. Bajo entry
- **Debilidad**: Menos maduro en MX que Facturama/CONTPAQi. Posicionamiento débil SAT compliance

### ❌ Kontafacil
- **Status**: ❌ NO ENCONTRADO en búsquedas web oficiales. Posible nombre alternativo, producto local desconocido, o discontinuado

### ❌ Nominaplus
- **Status**: ❌ "Sage NominaPlus" es para España (2012-2013). Sin producto MX específico de facturación

**Tagline**: *La única plataforma que integra Facturación + Crédito + Cobranza + Auditoría*

### VENTAJAS COMPETITIVAS CONFIRMADAS

| Vs. Competencia | Ventaja | Impacto |
|---|---|---|
| **Facturama** | WhatsApp auto + CobraCheck + Crédito | PyMEs → Medianas |

### RIESGOS A MONITOREAR

1. **Facturama**: ¿Mantiene API estable? ¿Aumenta precios? → Plan B: cambiar PAC
2. **CobraCheck adoption**: Si falla CobraCheck, FacturaCheck también cae → roadmap paralelo
3. **Competencia**: CONTPAQi podría lanzar WhatsApp + cheaper → mantener innovación
4. **Cumplimiento fiscal**: SAT cambia requisitos → equipo legal monitorea

### DIFERENCIALES QUE NO DEBE PERDER

✅ **WhatsApp automático** — Inversión prioritaria en distribución  
✅ **CobraCheck nativa** — No puede ser "bolted-on", debe ser arquitectura  
✅ **Crédito/sobregiro** — Monetización + retención cliente  
✅ **Precio híbrido** — Atrae ambos segmentos (fijo + destajo)

Si pierdes 2+ de estos, eres "Facturama con interfaz diferente". Mantén diferencial.
