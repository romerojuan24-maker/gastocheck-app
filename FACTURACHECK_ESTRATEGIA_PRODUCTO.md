#### Canal 2: Timbres Adicionales (Destajo)
```
Por cada timbre extra en plan básico: $4 (vs $5 competencia)
  → Atrae usuarios sensibles a precio
  → Incentiva upgrade a profesional cuando volumen crece

Híbrido (RECOMENDADO por usuario):
  Plan $399/mes = 100 timbres incluidos + $4 adicionales
  → Mejor TCO que cualquier competencia
```

#### Canal 3: Crédito & Sobregiro
```
Línea de sobregiro: +10-20% del saldo usado
  → Comisión: 2-3% del monto utilizado
  → Retención en siguiente pago
  → Atrae usuarios con urgencias (genera lealtad + ingresos)

Ejemplo:
  - Saldo: $1,000 MXN
  - Sobregiro disponible: $200 (20%)
  - Usuario usa $150
  - Comisión: 3% = $4.50
  → Siguiente periodo: $1,000 - $150 - $4.50 = $845.50 disponible
```

## 🎓 CRÍTICAS QUE EVITAR

### "Es solo un wrapper de Facturama"
**Defensa**: 
- No. Facturama = solo timbrado
- FacturaCheck = crédito + distribución + integración CobraCheck + reportes + auditoría fiscal
- Value-add 300% sobre Facturama puro

### "CONTPAQi es más robusto"
**Defensa**:
- Correcto para enterprise/despachos
- FacturaCheck para PyMEs que necesitan simple + móvil + whatsapp
- Precio 1/3 + UX 10x mejor + integración CobraCheck (CONTPAQi no tiene)

### "¿Qué si cambia el SAT?"
**Defensa**:
- Somos SaaS. Actualizamos instantáneamente
- Enterprise = esperar update software + reinstalar
- FacturaCheck = transparent updates, zero downtime

### "Competencia de Facturama"
**Defensa**:
- Somos **complemento** de Facturama, no competencia
- Facturama = PAC. FacturaCheck = aplicación + crédito
- ¿Analogía?: Stripe = procesa pagos. Shopify = plataforma que VENDE + procesa con Stripe
- Podemos ofrecer descuento Facturama a nuestros usuarios (win-win)

## 📌 DECISION FRAMEWORK

**Should we launch with X feature?**
- **YES if**: Core to MVP (CFDI, Facturama, WhatsApp, credit)
- **YES if**: Blocks launch (SAT compliance, security)
- **NO if**: Nice-to-have (white-label, API, mobile)
- **NO if**: Takes >1 week + not core (advanced reports, DIOT)

**Launch when**:
- MVP 100% complete ✅
- SAT audit passed ✅
- 50+ beta users tested ✅
- Support ready ✅
- Zero critical bugs ✅

**Don't launch when**:
- "Almost ready"
- "Will fix after launch"
- "User requested this"
- "Competitor has it"
